# SimplScenes — 4K Ambient Scenes for Apple TV

A tvOS app featuring beautiful ambient scenes with optional premium content packs.

## Features
- 10 free ambient scenes
- Premium scene packs via in-app purchase ($1.99-$2.99)
- Dark mode SwiftUI interface
- Focus-optimized remote navigation
- Music integration with Crafted Music House artists

## Scenes
### Free (10)
- Ocean Waves, Forest Rain, Fireplace, Northern Lights, Desert Sunset
- Mountain Stream, City Night, Cherry Blossoms, Thunderstorm, Starfield

### Premium (5)
- Arctic Aurora, Tropical Paradise, Space Nebula, Volcano Eruption, Ocean Shipwreck

## Technical Stack
- tvOS 17+
- SwiftUI
- StoreKit 2 (IAP)
- AVKit (video playback)

## Development
```bash
cd SimplScenes
swift build
```
